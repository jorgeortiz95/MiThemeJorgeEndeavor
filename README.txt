== Como puede ser montado ==
1. Instalar XAMPP
2. Crear una base de datos para WordPress teniendo en cuenta el nombre y usuario de la base de datos del archivo wp-config.php adjunto
	*Nombre de la Base de datos: teraning_wpdb3
	*Usuario: teraning_wpad3
	*Clave: MjQxYTI1MmM0MWM3NWEzNmY2ZDk4NGFl
3. Instalar WordPress en local con XAMPP
4. Pegar los archivos del theme MiThemaJorge en la ruta: localhost/wp-content/themes/
5. Diríjase a la opción de menú “Apariencia” en su panel admin de wordpress y Active el Tema MiThemaJorge
6. Listo ya el tema esta instalado y listo para ser revisado.
